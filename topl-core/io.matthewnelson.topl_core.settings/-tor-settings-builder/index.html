    TorSettingsBuilder¶

    class TorSettingsBuilder :CoreConsts (source)

    Call io.matthewnelson.topl_core.OnionProxyManager.getNewSettingsBuilder to obtain this class.

    This class is basically a torrc file builder. Every method you call adds a specific value to the buffer which Tor understands.

    You can call addLine if something isn’t covered here so you can customize your torrc file however you wish.

    Calling finishAndReturnString will return to you the String that has been built for you to write to the io.matthewnelson.topl_core_base.TorConfigFiles.torrcFile.

    Calling finishAndWriteToTorrcFile will do just that.

    onionProxyManager.getNewSettingsBuilder()
    .updateTorSettings()
    .setGeoIpFiles()
    .finishAndWriteToTorrcFile()
